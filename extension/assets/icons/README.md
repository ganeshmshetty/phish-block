# Extension Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16px - Toolbar icon (small)
- `icon32.png` - 32x32px - Toolbar icon (medium)  
- `icon48.png` - 48x48px - Extensions management page
- `icon128.png` - 128x128px - Chrome Web Store, installation dialog

## Icon Requirements

- **Format**: PNG with transparency
- **Design**: Shield logo with "Phish-Block" branding
- **Color scheme**: Match the app gradient (purple: #667eea to #764ba2)
- **States**: Consider creating variants for:
  - Default (protection active)
  - Disabled (protection off)
  - Warning (suspicious site)
  - Blocked (phishing detected)

## Creating Icons

You can use tools like:
- **Figma/Sketch** - Design all sizes
- **Image resizer** - Generate multiple sizes from one base image
- **Online generators** - favicon.io, realfavicongenerator.net

## Temporary Solution

Until proper icons are created, you can:
1. Use emoji-based icons as placeholders
2. Create simple colored squares with text
3. Generate from an online icon generator

The extension will still work without custom icons, but Chrome will show a default extension icon placeholder.
